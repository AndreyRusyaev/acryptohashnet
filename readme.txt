AcryptoHash.Net Project

This project is subproject for project Acrypto.Net.

Targets
Implementation of well-known cryptographic hash functions for .Net Framework.

Requirements
Crossplatforms for .Net Framework implementations (No depends from operational systems). Implementation in managed code followed from requirement above.

Tools
All algorithms from open sources, standards and request for comments (rfc).
For Testing purpose used NUnit (nunit.org).
